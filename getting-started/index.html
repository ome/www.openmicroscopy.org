Getting Started with OMERO

Whether you’re a new user with an institutional OMERO.server looking for help getting started, a developer wanting to read up on the nuts and bolts, or a system administrator wanting to install an OMERO.server, you’ll find the information you need by following the links below.

OMERO for Scientists

Your microscopy images securely stored but shareable and available from anywhere you have internet access.

From microscope to publications, OMERO handles all your images in a secure central repository. Work with your data from anywhere you have internet access using a desktop app, via your web browser or from 3rd party software. Organize, view and analyze your data, share it with colleagues, even make figures for publication and allow readers to link back to the original data. Over 140 image file formats supported, including all major microscope formats.

OMERO for Developers

Join the OME community and extend OMERO’s functionality to suit your individual needs.

As an open-source project, you’re free to write your own client code or extend the server to suit your needs. The OMERO API allows clients to be written in Java, Python, C++ or MATLAB and you can also add functionality via a scripting service. We manage all our code via GitHub, provide extensive documentation and are always happy to provide support via our mailing lists and forums.

Get started with OMERO for Developers →

OMERO for Your Institution

OMERO.server securely stores image data and enables users to manage, visualize, analyze and even publish all from the same platform.

With OMERO, all the image data from your facility can be securely stored and managed, using group permissions and user roles to allow controlled access tailored to your needs. From private repositories for sensitive data to hosting public data for your website or latest publications, OMERO is tried and tested in hundreds of institutions world-wide.
