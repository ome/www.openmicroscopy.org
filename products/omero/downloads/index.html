OMERO clients

A standard OMERO user just needs to download the client package with the same major version as their institutional server (e.g., 5.2.0 clients will connect to 5.2.5 servers but not to 5.3.0 servers). Full instructions for installation are on the help site.

If you do not have an institutional server, you can apply for an account on our demo server.
