OME-TIFF and OME-Zarr

What File Formats does OME develop?

The OME-TIFF file format was created to maximize the respective strengths of OME-XML and TIFF. It takes advantage of the rich metadata defined in OME-XML while retaining the pixels in multi-page TIFF format for compatibility with many more applications.

The OME-Zarr format is developed by the Next-generation file formats (NGFF) project, which is organised by OME. OME-Zarr is a cloud-optimized format designed to address scalability issues with large imaging datasets.
