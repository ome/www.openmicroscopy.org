Day 2 — Thursday 1st June, 2017 - Dalhousie Teaching Rooms

The workshops will feature a mixture of presentations and more informal discussion topics. Please feel free to approach a member of the team with your questions or to request a demonstration of the OMERO clients or specific functionality.

09:00 - 09:15

Arrival with tea/coffee

09:15 - 09:30

Introduction

09:30 - 10:45

Workshops - Round 1

  • Room 1 - Using OMERO
  • Informal demonstrations of OMERO.clients, especially OMERO.web
  • Informal demonstrations of web apps: figure, iviewer, webtagging etc.
  • Informal demonstrations of other tools developed by the OME consortium e.g. mtools, ImageJ
  • Discussions as requested on integrating your own tools: Java, Python etc.
  • Discussions as requested on using/customizing training materials
  • Room 2 - Working with OME Files and Bio-Formats (developer-aimed session)
  • Overview of OME Model and Formats, Bio-Formats and OME Files
  • Introduction to working with Bio-Formats
  • Introduction to working with OME Files
  • Small group breakout sessions
  • Room 3 - IDR annotations tour (please bring laptop)
  • Search for annotations in the IDR web user interface
  • Query and retrieve annotations using the web API
  • Annotation querying and analysis using Jupyter notebooks
  • Annotating your own images in OMERO (if time allows)
10:45 - 11:15

Tea/coffee & refreshments

11:15 - 12:30

Workshops - Round 2

  • Room 1 - OMERO DevOps / OMERO @scale
  • Introduction to OMERO @Scale
  • Tools for configuration, testing and deployment management (e.g. Docker)
  • Image Data Resource (IDR) case study (OpenStack, GPFS, public web, caching)
  • Room 2 - Import CLI
  • CLI import: advanced import and extensible import
  • Import options review, Import As
  • OME.smuggler: queue your imports (w/ Andrea Falconi, Montpellier)
  • Room 3 - Rocket: a flexible, modular, cloud-based platform for the management of heterogeneous information from acquisitions and tools for biomedical research (Chong Zhang, Neubias)
  • Introduction about the Rocket platform design architecture and technology
  • Use cases: 1. collaborative clinical data manager 2. bioimage analysis task manager
  • Demo and play
12:30 - 13:45

Lunch

13:45 - 15:00

Workshops - Round 3

  • Room 1 - OMERO for AWS (Douglas Russell, Harvard)
  • Introduction to Docker and AWS
  • Building archives locally with Docker
  • Dehydrating an archive to S3
  • Hydrating an archive in AWS with Cloud formation
  • Room 2 - Analyzing data in OMERO
  • Introduction to IDR and Jupyter
  • Analyzing data in OMERO using Python and R
  • Working through use-cases including image features
  • Room 3 - OMERO.web apps
  • Intro to OMERO.web framework
  • New JSON API
  • App overview: figure, iviewer, mapr, etc.
  • Available extension points
