Organisers: foundingGIDE

Location: RIKEN Institute, Kobe

Workshop 1: OMERO installation and configuration

Date: 4th November 2024

Duration: 2.5 hours

Workshop 2: OMERO imaging workflows

Date: 4th November 2024

Duration: 2.5 hours

Workshop 3: Advanced Analysis with OMERO

Date: 6th November 2024

Duration: 5 hours

OMERO, a component of OME, is a software platform for image data management and analysis.

Workshop 1: For the first time, the installation of OMERO.server and OMERO.web was shown in a workshop. The participants installed the components of the OMERO software in Ubuntu 22.04 Virtual Machines and learned how to install and configure the applications for OMERO.web, such as OMERO.iviewer.

Workshop 2: Import, organisation, viewing, searching, annotation and publishing of imaging data with OMERO were shown.

Workshop 3: Advanced analysis using OMERO API was shown. The emphasis was on the OMERO plugin for Fiji, but also the connection between ilastik and OMERO was explored.
