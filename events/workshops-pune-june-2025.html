--- layout: news-events filename: events title: June 2025 OME workshop main-blurb: The OME Team gives a 3-part workshop on OMERO at IISER (Pune, India). ---

Organisers: GBI

Location: IISER, Pune, India

Workshop Part 1: OMERO installation and configuration

Date: 19th June 2025

Duration: 4.5 hours

Workshop Part 2: OMERO imaging workflows

Dates: 19-20th June 2025

Duration: 6 hours

Workshop Part 3: Advanced Analysis with OMERO

Date: 20th June 2025

Duration: 1.5 hours

OMERO, a component of OME, is a software platform for image data management and analysis.

Workshop 1: The installation of OMERO.server and OMERO.web will be shown. The participants will install the components of the OMERO software in Ubuntu 22.04 Virtual Machines and learned how to install and configure the applications for OMERO.web, such as OMERO.iviewer. Further, the installation of OMERO using Docker images will be practiced.

Workshop 2: Import, organisation, viewing, searching, annotation and publishing of imaging data with OMERO will be shown.

Workshop 3: Advanced analysis using OMERO API wwill be shown. The emphasis wwill be on the OMERO plugin for Fiji, but also the connection between ilastik and OMERO will be explored.
