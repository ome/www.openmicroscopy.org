Organisers: GBI

Location: IISER Pune, India

Workshop: OMERO installation&configuration, Imaging workflows and Advanced Analysis with OMERO

Dates: 19-20th June 2025

Duration: 2 days

OMERO, a component of OME, is a software platform for image data management and analysis.

The installation of OMERO.server and OMERO.web was perfomed by the participants. The participants installed the components of the OMERO software in Ubuntu 22.04 Virtual Machines. The installation of OMERO using Docker images was practiced.

Further, import, organisation, viewing, searching, annotation and publishing of imaging data with OMERO was practiced.

As a further topic, advanced analysis using OMERO API was gone into. The emphasis was on the OMERO plugin for Fiji, but also the connection between CellPose and OMERO was explored.

The workshop finished with practical on how to make publication figures using the popular OMERO.figure application.
