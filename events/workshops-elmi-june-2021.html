--- layout: news-events filename: events title: June 2021 OME users workshops main-blurb: Three 1-hour workshops held online on OMERO, IDR and NGFF as part of the ELMI 2021 meeting. ---

Organisers: ELMI

Location: held remotely

Workshops:
Image Management, Sharing, Publication and Analysis in OMERO
Image Data Resource (IDR)
Next Generation File Formats

Dates: 23-25 June 2021

Duration: 1-hour per day for 3 days

The three workshops will focus on several key areas of OME Team's work covering data management of image data using OMERO open source data management solution, Image Data Resource (IDR) and cloud-based sharing of image data using next-generation image file format (NGFF). For more details please visit the workshops schedule and enter "Open" into the filtering box.
